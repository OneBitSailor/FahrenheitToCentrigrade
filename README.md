# Compose FahrenheitToCentrigrade
Demonstration of Fahrenheit to Centigrade (and converse) app built in Compose with utilizing Compose runtime ViewModel component.

## Features
 - Use strictly numeric android keyboard
 - Seperation of concerns implemented

## Tech Stack
 - Kotlin
 - Jetpack Compose
 - ViewModel
